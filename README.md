3DPLabelbot
A browser-based 3D label designer that exports print-ready STL files. Download .html and launch.

What it does:
Design embossed text labels for 3D printing. Adjust dimensions, font, text size, and colors, then export a binary STL ready for any slicer (PrusaSlicer, Cura, Bambu Studio).

Controls:

Text,
Font,
Dimensions,
Corner Fillet,
Text Depth,
Text Height

Export as an .STL
